# 🚀 AustroX-GPT

AustroX-GPT is a premium, AI-powered chat assistant built by **Nofal Fayyaz** — inspired by ChatGPT but tailored for speed, depth, and voice interaction.

🌐 Try it here: [https://nofalai.vercel.app](https://nofalai.vercel.app)

---

### ✨ Features

- 🔮 Quick & Deeper Thinking AI Modes
- 🎤 Voice Input Support
- 🌓 Dark/Light Mode Toggle
- 💎 Premium Diamond-Like UI
- 💬 Chat History with Auto Naming
- ✅ Mobile + Desktop Responsive
- 🔒 Local Login/Logout System
